------
install
--------
npm install "dgz-rn-pull-to-refresh" --save

------
base usage
----
```javascript
import PullView from 'dgz-rn-pull-to-refresh';
...
  render() {
        return (
            <View style={styles.container}>
                <PullView/>
            </View>
        );
    }
...   
```
see TestPullRefresh to learn more

-----
